## 개요

**Video Compressor Pro**는 영상 파일을 H.265(HEVC) 코덱으로 재인코딩하여  
파일 크기를 최대 97% 이상 줄여주는 GUI 압축 도구입니다.

### 실제 예시
| 원본 | 압축 후 | 감소율 |
|---|---|---|
| DJI 4K 드론 영상 1.7 GB | 43 MB | **97.5%** |
| 스마트폰 동영상 500 MB | 35 MB | **93%** |
| 강의 녹화 1 GB | 80 MB | **92%** |

## 특징

- ✅ **GUI 방식** — 직관적인 다크테마 인터페이스
- ✅ **파일 열기** — 여러 파일 동시 선택 압축
- ✅ **폴더 열기** — 폴더 내 모든 영상 일괄 압축 (하위폴더 선택 가능)
- ✅ **원본 삭제 옵션** — 압축 후 원본 보존 또는 삭제 선택
- ✅ **출력 경로 옵션** — 원본 폴더 내 `Comp/` 서브폴더 또는 직접 지정
- ✅ **실시간 진행률** — 파일별/전체 진행 바, ETA 표시
- ✅ **로그 패널** — 압축 결과 실시간 로그
- ✅ **ffmpeg 자동 설치** — 없으면 자동 다운로드 & 설치
- ✅ **EXE 빌드** — `build_exe.bat` 으로 독립 실행 파일 생성

## 실행 방법

### 방법 1 — 배치 파일 실행 (가장 간단)
```
setup.bat  더블클릭
```
- Python 자동 확인
- ffmpeg 자동 설치
- GUI 실행

### 방법 2 — Python 직접 실행
```bash
# 의존성 확인 (선택)
python installer.py

# GUI 실행
python main.py
```

### 방법 3 — EXE 파일 빌드 후 실행
```bash
# EXE 빌드 (최초 1회)
build_exe.bat  더블클릭

# 빌드된 EXE 실행
dist\VideoCompressorPro.exe
```

### 🎯 품질 (CRF)
| CRF 값 | 품질 | 파일 크기 | 권장 용도 |

| 18 | 거의 무손실 | 매우 큼 | 영상 편집용 원본 |

| preset | 속도 | 압축 효율 | 권장 |

## 압축 원리

Video Compressor Pro 는 다음 4가지 기법을 조합합니다:

### 1. 해상도 다운스케일
```
4K (3840×2160) → FHD (1920×1080)
픽셀 수 4분의 1로 감소, lanczos 필터로 고품질 축소
```

### 2. H.265 (HEVC) 코덱 재인코딩
```
H.265 는 H.264 대비 동일 화질에서 약 40~50% 작은 파일 크기
```

### 3. CRF (Constant Rate Factor)
```
화질을 일정하게 유지하면서 비트레이트를 자동 조절
움직임이 많은 장면은 높은 비트레이트, 정적 장면은 낮은 비트레이트
```

### 4. preset slow
```
인코딩에 더 많은 시간을 사용해 더 효율적인 압축 달성
같은 CRF 에서 ultrafast 대비 20~30% 작은 파일 크기
```

## 문제 해결

### ffmpeg 를 찾을 수 없습니다
→ GUI 에서 "🔍 ffmpeg 확인" 버튼 클릭 → 자동 다운로드 선택  
또는 `installer.py` 직접 실행

### 압축 후 파일이 없습니다
→ 출력 경로 확인 (원본 폴더 내 `Comp/` 서브폴더)  
→ 로그 패널에서 오류 메시지 확인

### EXE 빌드가 안 됩니다
→ `pip install pyinstaller` 먼저 실행  
→ 바이러스 백신이 차단하는 경우 예외 추가

### 압축이 너무 느립니다
→ preset 을 `fast` 또는 `medium` 으로 변경  
→ 또는 해상도를 낮춤
